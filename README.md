# ☕ Golden Coffee Landing Page

Uma landing page moderna e sofisticada desenvolvida para representar uma marca premium de café, combinando elegância visual, identidade forte e uma experiência imersiva para o usuário.

---

## 📸 Preview do Projeto

Interface contendo:

- Menu lateral interativo
- Navegação intuitiva
- Imagem de destaque em tela cheia
- Identidade visual personalizada
- Ícones para redes sociais
- Design inspirado em marcas de luxo

---

## ✨ Tecnologias Utilizadas

- **HTML5**
- **CSS3**

---

## 🎨 Características do Design

- Visual premium nas cores preto e dourado
- Layout moderno e responsivo
- Efeito de transparência (Glassmorphism)
- Tipografia elegante
- Elementos gráficos sofisticados
- Experiência visual imersiva

---

## 🚀 Objetivo do Projeto

Praticar técnicas de desenvolvimento front-end através da criação de uma landing page profissional, explorando conceitos de responsividade, posicionamento, estilização avançada e identidade visual.
